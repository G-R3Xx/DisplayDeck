# Contributing to DisplayDeck

Thanks for your interest in contributing to DisplayDeck.

DisplayDeck is a Windows utility focused on simple and reliable switching between desk monitor setups and TV gaming setups.

## Ways to contribute

You can help by:

- Reporting bugs
- Suggesting improvements
- Testing on different monitor / TV setups
- Improving documentation
- Improving display switching reliability
- Improving launcher app handling
- Improving the UI

## Reporting bugs

When reporting a bug, please include:

- Windows version
- GPU brand/model if relevant
- Number of connected displays
- Which displays are monitors and which are TVs
- Whether the TV is always connected or sometimes powered off
- What you expected to happen
- What actually happened
- Any error messages shown by DisplayDeck
