# Changelog

All notable changes to DisplayDeck will be documented in this file.

## [0.1.0] - 2026-05-23

### Added

- Native Windows display switching
- Desk Mode and TV Gaming Mode profiles
- Active display discovery
- Card-based modern UI
- Large TV Gaming Mode and Desk Mode buttons
- Global hotkeys:
  - `Ctrl + Alt + G` for TV Gaming Mode
  - `Ctrl + Alt + D` for Desk Mode
- Optional launcher app support
- Attempt to close launcher when returning to Desk Mode
- Minimize to system tray
- Tray menu:
  - Open DisplayDeck
  - Switch to TV Gaming Mode
  - Switch to Desk Mode
  - Exit DisplayDeck
- Start with Windows option
- Display switching delay notice
- Improved error message when a display is unavailable
- Custom app icon

### Notes

This is the first public release.
